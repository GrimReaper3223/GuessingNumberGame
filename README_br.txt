										***GuessingNumberGame v3.0_ALPHA;***

Jogo de adivinhação programado em Java na IDE NetBeans.


Funcionalidade:

Basicamente, o jogo consiste em adivinhar um número gerado aleatoriamente dentro de um intervalo específico, de acordo com o nível selecionado.
Para cada resultado errado, uma dica simples é gerada, fornecendo uma orientação ao usuário.


***Níveis:***

    - Se o nível selecionado for fácil, será gerado um número aleatório de 0 a 100;
    - Caso o nível selecionado seja intermediário, será gerado um número aleatório de 0 a 200;
    - Caso o nível selecionado seja difícil, será gerado um número aleatório de 0 a 500;
    - Se o nível selecionado for veterano, será gerado um número aleatório de 0 a 1000;
    - E se o nível selecionado for Personalizado, o usuário precisará configurar o nível antes de iniciar um jogo personalizado;
        |
        --- Se o usuário não configurar um nível, o jogo não iniciará devido à falta de dados básicos para executar o mecanismo de nível personalizado.
